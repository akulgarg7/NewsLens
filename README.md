## рџ’» Local Installation & Setup

To run this application locally, you will need Python 3.9+ installed on your machine.

**1. Clone the repository**
`git clone https://github.com/akulgarg7/NewsLens.git`
`cd NewsLens`

**2. Create and activate a virtual environment (Recommended)**
`python -m venv env`
On Windows:
`.\env\Scripts\activate`
On macOS/Linux:
`source env/bin/activate`

**3. Install the required dependencies**
`pip install -r requirements.txt`

**4. Configure your Environment Variables**
This project requires a free API key from [OpenRouter.ai](https://openrouter.ai/) to power the LLM analysis.
* Create a new file in the root directory named exactly `.env`.
* Add the following configuration to the file, replacing the placeholder with your actual key:

NARAROUTER_API_KEY=sk-or-v1-your-actual-api-key-here
NARAROUTER_BASE_URL=https://openrouter.ai/api/v1
NARAROUTER_MODEL_NAME=google/gemma-4-26b-a4b-it:free

**5. Launch the Dashboard**
`streamlit run app.py`

*The application will automatically open in your default web browser at `http://localhost:8501`.*
